## Login Troubleshooting Playbook

### Symptoms
- Invalid username or password error
- Login page refreshes without logging in
- Login button unresponsive

### Possible Causes
- Cached credentials in browser
- Incorrect saved password
- Account temporarily locked
- Browser compatibility issue

### Resolution Steps
1. Confirm correct username and email
2. Clear browser cache and cookies
3. Attempt login in an incognito/private window
4. Reset password and retry login
5. Test using a different browser if needed

### Escalation Criteria
- Account locked after multiple attempts
- User unable to reset password
- Multiple users reporting the same issue
