## App Crash Troubleshooting Playbook

### Symptoms
- App crashes on launch
- Freezes at startup
- Error dialog appears

### Possible Causes
- Corrupt cache
- Outdated version
- OS incompatibility
- Permission issues

### Resolution Steps
1. Restart device
2. Update OS and app
3. Clear cache
4. Repair install
5. Reinstall app

### Escalation Criteria
- Persistent crash after reinstall
- Widespread reports
- Server-side dependency suspected
